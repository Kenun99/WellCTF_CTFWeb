# WellCTF_CTFWeb
CTF在线评测流程

欢迎提交issues
详情联系chenwm_@outlook.com

环境：Django2.1+python3.5+Bootstrap3+pycharm

功能：
  用户系统：
          登陆注册系统、用户修改密码和个人信息系统、/admin/后台更改信息。
  题目系统：
          支持除web题目意外的所有题型，在题目弹出的模态框下载文件再提交flag即可完成提交。
  比赛系统：
          比赛系统支持根据比赛时间确定是否进行比赛
  战绩系统：
          可以查看用户自己成功完成的题目
  Rank系统：
          用户可以查看top10用户
